Configuração:

1. Extraia a pasta do arquivo "SiteJC.rar";
2. Dentro do diretório, abra o aquivo "index.html" para acessar o site.

Estrutura:

O site foi dividido em 4 partes:
- O arquivo "index.html" é a página inicial do projeto;
- O diretório "img" contém as imagens usadas para o projeto.
- O diretório "pags" contém as outras páginas HTML usadas no projeto.
- O diretório "css" contém o arquivo "estilo.css" que foi usado para a estilização do projeto.

Funcionalidade:

O site trata-se de uma página de homenagem ao cantor americano Johnny Cash, contendo informações da biografia, discografia, cadastro para acompanhamento do site, e ainda sugestões para os organizadores através da navegação. Sua função é dispor ao usuário conteúdo informativo a respeito de seu ídolo.
